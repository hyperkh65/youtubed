# 🎨 KeyPoint Pro UI/UX 재설계 계획

## 개요
YouTube 중심 키워드 분석 도구 → **범용 멀티인더스트리 키워드 분석 플랫폼**으로 전환

---

## 📱 주요 변경사항

### 1. 헤더/네비게이션
**기존**: "유튜브 마케팅을 위한 가장 강력한 키워드 분석 도구"
**변경**: "모든 산업의 키워드 기회를 찾아주는 데이터 분석 플랫폼"

```
┌─────────────────────────────────────────┐
│ KeyPoint Pro (로고)    | 로그인 | 가격책정 │
├─────────────────────────────────────────┤
│ 검색 바 + 산업 선택                       │
│ 포털 선택 (Google/Naver/Daum/YouTube) │
└─────────────────────────────────────────┘
```

### 2. 산업 선택 UI (신규)
4가지 주요 산업별 분석 모드 제공:
```
┌──────────────────────────────────────────┐
│  🔍 어떤 산업의 키워드를 분석하시나요?    │
├──────────────────────────────────────────┤
│ ┌─────────────┐  ┌─────────────┐        │
│ │ 📈 SEO &    │  │ 🛒 이커머스  │        │
│ │    블로그   │  │    분석     │        │
│ └─────────────┘  └─────────────┘        │
│ ┌─────────────┐  ┌─────────────┐        │
│ │ 🎬 콘텐츠   │  │ 🏢 마케팅   │        │
│ │    크리에이터 │  │    대행사   │        │
│ └─────────────┘  └─────────────┘        │
└──────────────────────────────────────────┘
```

**산업별 최적화 내용:**

#### SEO & 블로그
- 백링크, 도메인 권위도 추가 (향후)
- 관련 키워드 클러스터
- 콘텐츠 갭 분석

#### 이커머스
- 상품 판매량 트렌드
- 리뷰 평점, 가격 경쟁도
- 쿠팡/Amazon 통합

#### 콘텐츠 크리에이터
- 트렌딩 지표 강화
- 영상 추천검색어
- 바이럴 가능성 점수

#### 마케팅 대행사
- 다중 계정 분석
- 경쟁사 보고서 (자동)
- 월간 리포트 기능

### 8. 가격/구독 정보 통합

#### 무료 vs Pro 기능 표시
```
Free Tier (10분석/월):
├─ 단일 포털 분석
├─ 기본 메트릭만 (검색량, 난이도)
├─ 신뢰도: 🔴 50-70% (제한)
└─ 월 1회 포털 비교

Pro Tier ($19/월):
├─ 무제한 분석
├─ 모든 포털 동시 분석 (6개)
├─ 고급 메트릭 (의도, 계절성, 경쟁사)
├─ 신뢰도: 🟢 85%+ (Google API)
└─ 월별 자동 리포트
```

  if (score >= 85) color = 'green' // 🟢
  else if (score >= 70) color = 'yellow' // 🟡

  return (
    <span className={`bg-${color}-500 px-2 py-1 rounded text-xs`}>

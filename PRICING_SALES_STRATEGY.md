## 📊 가격 구조

### 1. Free Tier (무료)
**월 가격**: $0
**목표**: 신규 사용자 확보 & 제품 체험

#### 기능
```
✅ 월 10회 분석
✅ 단일 포털 분석만 (Google, Naver, Daum, YouTube 중 1개)
✅ 기본 메트릭만 (검색량, 난이도)
✅ 신뢰도: 🔴 50-70% (AI 추정값만)
✅ 관련 키워드 제한 (3개만)
❌ 검색 의도 분석
❌ 경쟁사 분석
❌ 12개월 트렌드
❌ 뉴스/블로그 추적
❌ CSV 다운로드
❌ API 접근
```

#### 제한 로직
```typescript
const canAnalyze = userTier === 'free'
  ? monthlyAnalysisCount < 10
  : true

const portalsAvailable = userTier === 'free'
  ? selectedPortals.slice(0, 1)
  : selectedPortals

const trustBadgeScore = userTier === 'free' ? 60 : 85
```

**월 가격**: $19 USD
**목표**: 활동 사용자 수익화
**예상**: 750명 × $19 = $14.25K/월 ($171K/년)

#### 기능
```
✅ 무제한 분석
✅ 6개 포털 동시 분석 (Google, Naver, Daum, YouTube, Amazon, Coupang)
✅ 모든 메트릭 (검색량, 난이도, CPC, 기회점수)
✅ 신뢰도: 🟢 85%+ (Google API + 공식 파트너 데이터)
✅ 관련 키워드 무제한
✅ 검색 의도 분석
✅ 경쟁사 분석 (5개)
✅ 12개월 트렌드 분석 & 계절성
✅ 뉴스/블로그 30일 추적
✅ CSV/PDF 다운로드
✅ 월별 자동 리포트
✅ 우선 지원 (이메일)
❌ API 접근
❌ 다중 계정
```

#### 비용 분석
```
데이터 비용: $7.5K/월
수익: $19 × 750 = $14.25K/월
순이익: $6.75K/월 (47.3%)
```

## 💡 판매 전략

### 1. Free → Pro 전환 (주요 매출)

#### 전환 시나리오
```
신규 사용자 (1000)
  ↓ (10% 무료 체험)
활동 사용자 (100)
  ↓ (월 10회 제한으로 Pro 필요)
유료 사용자 (75) = 75명 × $19 = $1,425/월
```

#### 전환 트리거
1. **월 10회 분석 도달** → "더 분석하려면 Pro로 업그레이드"
2. **포털 제한** → "6개 포털 동시 분석은 Pro 기능"
3. **고급 기능 클릭** → "이 기능은 Pro에서만 사용 가능"
4. **검색 의도 탭** → Pro 배지 + "Pro로 업그레이드" 버튼

```

```

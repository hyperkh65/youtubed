# 📊 범용 키워드 분석 플랫폼 - 완전한 기획안

## Executive Summary

**제품명**: KeyPoint Pro (KPP)
**타겟**: 마케팅담당자, SEO전문가, 이커머스운영자, 콘텐츠크리에이터
**포지셔닝**: "모든 산업의 키워드 기회를 찾아주는 데이터 분석 플랫폼"
**경쟁우위**: Black Kiwi + Semrush의 깊이 + 실시간 다중포털 분석

## 1️⃣ 시장 분석 & 기회

### 시장 규모
- **글로벌 SEO/마케팅 도구 시장**: $80B (2024)
- **한국 시장**: ~$2B (5개 주요 기업 점유)
- **연평균 성장률**: 15-20%

### 경쟁사 분석

| 제품 | 강점 | 약점 | 가격 |
|------|------|------|------|
| **Black Kiwi** | 한글 최적화 | 단일 포털(Naver) | $29/월 |
| **Semrush** | 글로벌 데이터 | 높은 가격 | $139/월 |
| **Ahrefs** | 백링크 분석 | 진입 어려움 | $99/월 |
| **SE Ranking** | 가성비 | 기능 제한 | $39/월 |
| **KPP (우리)** | **다중포털 + 실시간** | 신규브랜드 | $19/월 |

### 시장 기회
✅ **B2C**: 개인 크리에이터, 블로거 (무료→프로 전환)
✅ **B2B**: 마케팅 대행사, 이커머스 회사 (대량 분석)
✅ **B2B2C**: 웹호스팅, CMS 플랫폼 통합
✅ **수직통합**: 산업별 특화 (의료, 이커머스, SaaS 등)

## 2️⃣ 제품 전략

### 2.1 핵심 기능 (MVP)

#### 1️⃣ 다중포털 키워드 분석
```
지원 포털:
- Google (전 세계)
- Naver (한국)
- Daum (한국)
- YouTube (영상)
- Amazon (이커머스)
- 쿠팡 (한국 이커머스)

핵심 메트릭:
✓ 검색량 (월간/주간/일일)

✓ CPC / 트렌드
✓ 계절성 + 변동성
```

```
🎯 검색 의도 분석
├─ Informational: 교육 콘텐츠
├─ Commercial: 비교/리뷰
├─ Transactional: 구매 유도

💼 산업별 최적화
├─ SEO/블로그: 백링크, DA/PA
├─ 이커머스: 판매량, 가격대, 리뷰
├─ SaaS: 비즈니스 가치, 의도 강도
└─ 콘텐츠: 트렌드, 바이럴 가능성

🏆 경쟁사 분석

├─ 우리만의 기회 키워드

└─ 차별화 전략
```
